## Using the webapp
This project uses [Pipenv](https://pipenv.readthedocs.io/en/latest/). So start with to create a virtual folder on your machine.

```
pipenv install
```

Afterwards, use:

```
pipenv shell
```

And if you want to exit the env, use:

```
exit
```
